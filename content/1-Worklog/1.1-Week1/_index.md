### Goals for week 1: (Exploration Phase)
* Take on the Leader role, get to know everyone, and form a group for the bootcamp internship.
* Hold meetings to brainstorm detailed ideas for the EduTrust project, and unify everyone's opinions on the core feature developments.
* Form a project team and agree on choosing the EduTrust topic.
* Set up a team working environment via Git, Jira.
   * Use a Git Monorepo model to easily manage and sync both Backend (FastAPI) and Frontend (Next.js/React).
   * Apply Agile on Jira, creating concise Epics and Tickets for the 5 members to prevent task conflicts.
* Conduct preliminary research on Cloud services.
   * Understand the cloud learning roadmap.
   * Open an AWS account.
   * Learn about AWS basic services and practice using them.
   * Practice strict IAM permissions (restricted Access Keys) to ensure the Backend safely interacts later.
